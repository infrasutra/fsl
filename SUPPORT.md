# Support

## Where to Ask Questions

Use GitHub Discussions for usage questions, architecture ideas, and
troubleshooting that is not a confirmed bug.

## Where to Report Bugs

Use GitHub Issues for reproducible bugs and regressions.

Please include:

- Your command and input schema snippet
- Expected behavior and actual behavior
- `go version` and OS details
- Minimal reproduction steps

## Security Issues

For vulnerabilities, do not file a public issue.
Follow [SECURITY.md](SECURITY.md).
